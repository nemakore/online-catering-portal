## ⚙️ Installation & Setup

### 🔽 Step 1: Clone Repository

```bash
git clone https://github.com/nemakore/online-catering-portal.git
```

### 📂 Step 2: Move to XAMPP

```bash
Copy folder → C:/xampp/htdocs/
```

### ▶️ Step 3: Start Server

* Start Apache & MySQL from XAMPP

### 🗄️ Step 4: Setup Database

1. Open **phpMyAdmin**
2. Create a new database
3. Import `.sql` file from project

### 🌐 Step 5: Run Project

```
http://localhost/catering
```
